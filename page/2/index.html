引言

今天的信息可视化课程并不是十分顺利。
这不是由于老师讲得太快自己没有跟上，而是因为自己不在状态且参与课程的大佬太多了，弱弱的自己显得很无助。
##课程相关细节
###考核标准
10%出勤及课堂表现+40%上机练习和课后作业+50%期末项目

  1. 期末项目是做一个数据可视化的系统。这个在下周五（7月15日）的课堂上要用自己的ppt做展示，而不是在那时才布置作业然后之后留着完成。该项目大致是利用相关的数据，做出很好的交互项目。可以组队，每组不超过2人。
  2. 老师上课会随机地抽人回答问题或者在那个超星上面蛇皮签到，这个算在第一笔成绩，要注意！第二笔成绩就是超星上每天提交的项目。
  3. 这个课每天是4节，前2节讲理论，后两节讲实践。
    ###介绍
  4. 资料一部分在老师主页还有一部分在超星学习通（要进入旧版本，然后课程资料才能看得见）
  5. 可视化是用可视形式进行解释的动作或过程。
  6. 本课程可能涉及的编程语言？JavaScript (与C/C++语法类似，但是弱类型，比较好写和灵活)
  7. 可视化这个方向十分深奥，也有很大的应用前景，在日后一定有很大实用价值，信息可视化这个课程只是入了个门。
  8. ECharts 是一个使用 JavaScript 实现的开源可视化库，涵盖各行业图表，满足各种需求。课程主要在用Echarts(入门比较简单)（python插件pyechart)。另外地，d3比较牛，但是课上讲的少。
  9. Pandas是一个开源的Python库，用于使用其强大的数据结构进行高性能数据处理和数据分析。使用Pandas，无论数据源如何，我们都可以用于数据处理和数据分析工作。老师还推荐了一本书《web scraping with python》，这本书是关于利用python构建网络爬虫的。

C语言学习笔记1

发表于 2022-06-22 更新于 2023-01-11 分类于 lecture_study

前言

Hello there!
好久不见，这是暑假以来自己重拾博客的第一篇技术笔记。
由于下学期学习计算机系统（csapp）之必要，重学c语言，打下一个坚实的基础显得尤为必要。我在正式开始预习乃至学习这么大课之前，也希望自己去突破一些自己之前不曾注意的重要问题。
好了，就让我们开始吧。
《C程序设计语言》by 布莱恩W克尼汉& 丹尼斯M里奇
就是那本薄黑书:)

类型运算符和表达式

  1. 变量名是字母和数字组成的序列，第一个字符一定是字母。(此处下划线认为是字母且大小写字母有区分)
  2. 目前的编译器支持最大的变量名称长度为31
  3. 各个基本类型的字节数 1字节=8位 可以用sizeof()查询
    char 1 short 2 int 4(旧编译器是2) long 4 float 4 double 8
    long double 16
    两种形式 unsigned signed 前者不带负数可以扩大正数的范围
  4. unsigned/signed 可用于限定整形和char. unsigned总是正值或0且遵守算数模2^n定律.unsigned char的范围是0-255 signed char 为-128-127 不注明的char看机器，但是可打印字符总是正值
  5. long类型常量以l或L结尾如12345678L 无符号常量以u或U结尾 两者结合，就是ul或UL结尾
  6. 没有后缀的浮点数常量为double类型，后缀f/F表示float类型
  7. 前缀为0的整数常量表示八进制 前缀0x或0X表示十六进制。十进制37八进制037十六进制0x1F
  8. 一个字符常量是一个整数,放在单引号里面。比如’0’对应了整数48
  9. 转义字符 \a响铃 \b回退符 \f换页 \n换行 \t横向制表 \v纵向制表 \r回车 \反斜杠 ?问号 '单引号 '‘双引号 \ooo八进制 \xhh十六进制 ‘\0’表示值为0的字符即null
  10. #define常量表达式在编译时求值而不是运行时，可以出现在常量可能出现的任何位置。其实常数表达式也是编译时求值,比如1+1
  11. 一个说明双引号不是字符串的例子 char c[]=”456” “123”;等价于c[]=”456123” 要想在字符串中表示双引号，加个\转义即可
  12. 虽然看不见，字符串的内部以’\0’空字符结尾，因此字符串的存储空间比括在双引号之中的字符长度多1
  13. enum枚举类型是一个常数整型值的列表 enum test {a=0,b=1,c,d};其中c d的值分别为2 3自动生成了 enum和#define很相似但是优点在于可自动生成
  14. 所有变量必须先声明后使用，一个声明指定了一种变量类型。int a=3,b; 声明 初始化一起干 const char msg[]=”warning”;加上const限定符后该变量或数组的值不能修改啦 如果这样f(const msg[])表示f函数无法修改msg数组元素的值
  15. 运算符的优先级(这个上网查比较全，%和* / 一个优先级；位运算优先级不如+- )取模运算符不能用于浮点型
  16. EOF是文件结尾的标志，即整个文件或标准输入的结尾 而’\0’是字符串的结尾
  17. 类型转换之自动转换 将范围小的转换为范围大的 比如a+b其中a是整数b是浮点数 然后a的值会被转换为浮点数才与b的值相加.强制类型转换比如转化为int类型 (int)b; 大范围到小范围的会把超出的位舍弃
  18. ++n 和 n++效果不同前者是先+1后使用，后者是先使用后+1
  19. 位运算符 &按位与 |按位或 ^按位异或 <<左移 >>右移 按位取反 比如360 按位与常用于屏蔽某些二进制位 x<<2 x左移2位
  20. x*=y+1的含义是x*(y+1)而不是x*y+1 a op b的含义是 (a)op (b)
  21. 赋值语句具有值,这个值就是被赋的值。 比如while((c=getchar())!=EOF )……
  22. 条件表达式 (条件) ? a: b 条件成立则返回a 否则用b

AES简介

AES加密算法即密码学中的高级加密标准（Advanced Encryption Standard，AES），又称Rijndael加密法，是美国联邦政府采用的一种区块加密标准。这个标准用来替代原先的DES，已经被多方分析且广为全世界所使用。经过五年的甄选流程，高级加密标准由美国国家标准与技术研究院 （NIST）于2001年11月26日发布于FIPS PUB 197，并在2002年5月26日成为有效的标准。

AES算法本质上是一种对称分组密码体制，采用代替/置换网络，每轮由三层组成：线性混合层确保多轮之上的高度扩散，非线性层由16个S盒并置起到混淆的作用，密钥加密层将子密钥异或到中间状态。AES是一个迭代分组密码，其分组长度和密钥长度都是可变的，只是为了满足AES的要求才限定处理的分组大小为128位，而密钥长度为128位、192位或256位，相应的迭代轮数N，为10轮、12轮、14轮。AES汇聚了安全性能、效率、可实现性、灵活性等优点。最大的优点是可以给出算法的最佳查分特征的概率，并分析算法抵抗查分密码分析及线性密码分析的能力。

KMP字符串匹配

发表于 2019-06-17 更新于 2023-01-08

KMP算法可在一个主文本串 S 内查找一个词 W 的出现位置
此算法通过运用对这个词在不匹配时本身就包含足够的信息来确定下一个匹配将在哪里开始的发现，从而避免重新检查先前匹配的字符。

朴素

有一个文本串S，和一个模式串P (两者长度分别为N,M)

如果用暴力匹配的思路，并假设现在文本串S匹配到 i 位置，模式串P匹配到 j 位置，则有：
如果当前字符匹配成功（即S[i] == P[j]），则i++，j++，继续匹配下一个字符；
如果失配（即S[i]! = P[j]），令i = i - (j - 1)，j = 0。相当于每次匹配失败时，i 回溯，j 被置为0。
