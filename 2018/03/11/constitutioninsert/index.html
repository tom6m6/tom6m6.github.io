谈2018宪法修正案

发表于 2018-03-11 更新于 2018-07-31

2018两会，注定不寻常。
提出了备受争议的宪法修正案并成功通过。

修正案内容包括赋予地级市地方立法权、设立国家监察机关、取消国家主席和国家副主席任期限制、从宪法层面上确立宪法宣誓制度。

回顾

2月25日，政府喉舌新华网发表了《中国共产党中央委员会关于修改宪法部分内容的建议》 引发了大量网友的热议。
3月5日，第十三届全国人民代表大会第一次会议开幕。当日，代表大会第一次全体会议听取了全国人大常委会副委员长兼秘书长王晨关于中华人民共和国宪法修正案草案的说明。
3月7日，代表团全体会议、代表团小组会议分别审议宪法修正案草案。
3月11日下午，代表大会第三次全体会议投票表决通过中华人民共和国宪法修正案草案，现场收到赞成票2958张，反对票2张，弃权票3张，无效票1张。

意味

  1. 删除任期限制，意味着领导人（习近平）可以无限期执政
  2. 新设立的监察机关，使监察的范围更广。在反腐的同时，便利了打击政敌

摘录《国家监察法》
第三章 监察范围

第十二条 监察机关按照管理权限对下列公职人员进行监察：
（一）中国共产党的机关、人大机关、行政机关、政协机关、监察机关、审判机关、检察机关、民主党派和工商联机关的公务员及参照《中华人民共和国公务员法》管理的人员;
（二）法律、法规授权或者受国家机关依法委托管理公共事务的组织中从事公务的人员;
（三）国有企业管理人员;
（四）公办的教育、科研、文化、医疗卫生、体育等单位中从事管理的人员;
（五）基层群众性自治组织中从事集体事务管理的人员;
（六）其他依法履行公职的人员。

原因

背景

再谈原因之前，我们必须要了解一下中共内部的“隔代指定”规则。
最初，这个规则由邓制订，目的是为了杜绝终身制和避免任命亲信为下一代领导人。比如江(第3代)指定习(第5代)，胡(第4代)指定第六代，依此类推。
事实上，胡已指定了胡春华作为第6代掌门人(总书记)，孙政才作为他的搭档(总理)
理论上来讲，胡孙两人应该在十九大当选政治局常委，但是现实却不是如此。
在去年十九大召开前，孙政才被双开，胡春华只是保留了政治局委员的身份，没有升级。
这意味着十九大没有产生接班人。

分析

综合习近平最近几年对政敌的严厉打压，原本的派系，江派，胡派损失惨重。这固然发展了习的势力，但是新兴的的习派势力充满了不稳定因素。现在习处于一个骑虎难下的境地。如果他下台，那么又一个势力把他经营几年的势力消灭了怎么办？
针对胡孙两人的打击，是为可靠之人腾出位置。我认为，习近平不会幻想像皇帝一样统治到死。这样，必然会遭到党内的其他势力甚至战友的反对。
在合适的人选方面，他必然是犹豫的。他需要时间去筛选，他需要时间来巩固势力。这一切，可能是在时间上的缓冲。

影响

  1. 综合上面的论述，这为习换来了时间
  2. 习派的发展壮大，江派胡派的没落，是大势所趋。在未来10年内的范围内，这样有利于整体的稳定。从而避免了出现大规模政治冲突的可能
  3. 习近平不会一直执政。在条件满足后他会全身而退。但是，如果是其他人呢？未来，如果真的出现一个这样的人，对权力的野心超过了盟友的背叛。那么会造成什么？政局会急转直下，混乱无法避免
  4. 对普通的吃瓜群众来说，这件事几乎没有影响。除非他们显著感到了生活的不同。

观点

  1. 政治是险恶的。习近平会背上”独裁”的骂名，但是，这也比滑入深渊成为政治斗争的失败者强。任何事的发生必然有原因，他这样做也是有原因的。
  2. 对政治的思考需要谨慎和理性。政治，是一种特殊环境下人性的代表，作为旁观者，我们无法完全理解那一切。不能因为集体的狂热而丧失自己的主观思考
  3. 稳定，是整个体系的归宿
